# Frontend Architecture

## Tech Stack
- React 20 + Vite
- TypeScript
- Recoil or Redux for state management
- Chart.js / Recharts for graphs
- TailwindCSS for UI

## Structure
- `components/` - reusable components
- `pages/` - dashboard, settings, reports
- `services/` - API calls
- `hooks/` - custom React hooks
- `store/` - state management (user, device, billing data)
