# Backend Architecture

## Tech Stack
- Java 17 + Spring Boot
- PostgreSQL
- Docker
- Redis (optional)
- MQTT broker for IoT device communication

## Modules
1. **Device Management**
2. **Energy Data Ingestion**
3. **Billing & Pricing**
4. **User Management**
5. **Analytics & Reports**

## Notes
- Designed for multi-region support
- Future-proof for custom devices beyond Tuya
