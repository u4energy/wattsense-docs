# ⚡ Wattsense Documentation

Smart Energy Monitoring & Cost Intelligence Platform

---

## 📖 Pengenalan

Wattsense ialah platform pemantauan dan pengurusan tenaga pintar yang dibangunkan oleh U4 Energy untuk membantu pengguna memantau penggunaan elektrik secara masa nyata, mengawal peranti dari jauh, menganalisis penggunaan tenaga serta mengoptimumkan kos elektrik.

Repository ini berfungsi sebagai pusat dokumentasi rasmi bagi semua maklumat berkaitan produk, perniagaan, pelanggan, seni bina sistem, integrasi, operasi dan pembangunan Wattsense.

---

## 🎯 Objektif Repository

Repository ini diwujudkan untuk:

* Menjadi sumber rujukan tunggal (Single Source of Truth) bagi produk Wattsense.
* Menyimpan dokumentasi teknikal dan perniagaan secara berpusat.
* Membantu proses onboarding ahli pasukan baharu.
* Mendokumentasikan keputusan produk dan teknikal.
* Memastikan pengetahuan projek tidak bergantung kepada individu tertentu.

# 🚀 Overview

Wattsense membantu pengguna memantau penggunaan elektrik (kWh) dan menukarkannya kepada kos sebenar (RM) secara masa nyata.

Platform ini dibangunkan bagi meningkatkan keterlihatan penggunaan tenaga serta membantu pengguna membuat keputusan yang lebih tepat untuk mengurangkan kos elektrik.

---

# 🎯 Problem

Kebanyakan pengguna hanya mengetahui jumlah penggunaan elektrik selepas menerima bil bulanan.

Keadaan ini menyebabkan:

* Tiada keterlihatan penggunaan tenaga secara masa nyata.
* Sukar mengenal pasti pembaziran tenaga.
* Tiada kawalan terhadap kos penggunaan elektrik.
* Tiada data yang mencukupi untuk tujuan ESG dan sustainability.

---

# 💡 Solution

Wattsense menyediakan platform berpusat yang membolehkan pengguna:

* Pemantauan penggunaan tenaga secara masa nyata.
* Penukaran penggunaan kWh kepada nilai RM.
* Kawalan peranti secara jarak jauh.
* Pengesanan penggunaan tenaga luar biasa.
* Analisis dan laporan penggunaan tenaga.
